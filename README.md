## About

This is a console game - "Brain-games". The game has 5 different mini-games: brain-even, brain-calc, brain-gcd, brain-progression, brain-prime.

## Installation

```bash
make install
npm link
```

## Running

```bash
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
